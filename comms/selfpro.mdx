## Hiring

The [hiring](https://discord.com/channels/1356341862486507743/1435774015242244207) channel is for servers that are currently hiring. Breaking rules of this channel will result to 2 hour timeout on first offence.

<Warning>
  You must use the format (below)
</Warning>

```
Server Name: __________
Server Invite: _________
Positions Hiring: ________
```

## Numbered Channels

Channels that are numbered in this category are channels **with **that amount of members. 

<Warning>
  Please only post your invite one time per 5 minutes. Not following this rule will result in a 2 hour timeout on first offence. Format is below.
</Warning>

```
These channels do not require a format.
```
